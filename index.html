x100 Bot Commands

Prefix: x100. | Timezone: Central Standard Time (CST)

Giveaway Setup

  • x100.set_giveaway <Month> <Day> <Time> <AM/PM> <MaxGemsInMillions> <GemsPerEntryInMillions> <Prize Name> Sets the giveaway end date/time, gem caps, and prize.
    All times are in Central Standard Time (CST).
    Example: x100.set_giveaway June 15 8:30 PM 500 50 Prize

Leaderboard

  • x100.post_leaderboard Posts the giveaway leaderboard message (admin only).
  • x100.leaderboard Manually refreshes the leaderboard.

Donation Management

  • x100.scan_donations Scans the donation channel for donation messages and updates the leaderboard.
  • x100.entries <username> <entry_count> Manually add entries for a user based on gem value.
    Example: x100.entries exampleuser 3

Remove/Reset

  • x100.remove <username> Removes a user and their donation from the leaderboard.

How to Set Max Gems or Gems Per Entry

You can customize the donation cap and entry value for any giveaway using thousands, millions, or even billions of gems! All values in the x100.set_giveaway command are entered in millions.

Setting Small Values (Thousands)

To set gem values in the thousands, use decimals:
0.25 = 250,000 gems
0.8 = 800,000 gems
0.05 = 50,000 gems

Example:
x100.set_giveaway June 15 8:30 PM 0.8 0.05 Small Prize
This sets:
Max Gems = 800,000
Gems per Entry = 50,000

Setting Medium Values (Millions)

To set gem values in the millions, use whole numbers:
1 = 1,000,000 gems
50 = 50,000,000 gems
500 = 500,000,000 gems

Example:
x100.set_giveaway June 15 8:30 PM 500 50 Epic Prize
This sets:
Max Gems = 500 million
Gems per Entry = 50 million

Setting Large Values (Billions)

To go above 1 billion, use numbers over 1000:
1000 = 1,000,000,000 gems
1500 = 1.5 billion
5000 = 5 billion

Example:
x100.set_giveaway June 15 8:30 PM 5000 1500 Legendary Prize
This sets:
Max Gems = 5 billion
Gems per Entry = 1.5 billion

This system gives you full control to create giveaways of any size from tiny community events to massive high-roller competitions!

Rules & Info

  • 1 entry = the gem value you set in x100.set_giveaway (e.g. 50 million)
  • Max gems counted = the limit you set in x100.set_giveaway (e.g. 500 million)
  • Max entries = calculated automatically based on your settings
  • Only users with at least 1 entry appear on the leaderboard
  • Display shows gems counted toward entries, not excess
